# 零售数据分析报告（简单版）

## 我发现的主要情况

### 1. 业务增长很快
- **2010年**：卖了49.7万元
- **2011年**：卖了217.7万元  
- **增长率**：338%（翻了3倍多）

### 2. 客户变多了
- **2010年**：611个客户
- **2011年**：1,884个客户
- **增长**：208%（客户翻倍）

### 3. 有两种赚钱方式
**方式一：批发大客户**
- 有一个客户一次买了16.8万元的纸艺产品
- 这种客户虽然少，但金额很大

**方式二：零售小客户** 
- 大部分客户每次买几百几千元
- 有一个忠实客户一年买了14次

### 4. 商品卖得好的
1. 纸艺产品：16.8万元
2. 邮费服务：6万元
3. 蛋糕架：4.6万元

### 5. 主要在英国卖
- 英国：占了95%的销售额
- 德国、法国：各占2%

### 6. 质量不错
- 退货率只有1.85%
- 说明产品质量和服务都挺好

## 我的建议

1. **照顾好大客户**：那个买16.8万的客户要重点维护
2. **保证畅销品供应**：纸艺产品要多备货
3. **试试其他市场**：可以在德国法国多卖一些

## 我学到了什么

- 用SQL查数据挺有用的
- 看数据要多个角度一起看
- 数字背后都有业务含义
