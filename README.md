#  零售业务数据分析与洞察（2010-2011）

一个完整的数据分析项目，展示了从 **原始数据清洗 → SQL深度分析 → 业务可视化 → 商业策略建议** 的全流程。本项目旨在模拟数据分析师如何通过数据驱动的方式，为零售业务诊断健康度、识别增长机会。

##  项目亮点

- **闭环分析流程**：完整覆盖数据分析各环节，体现从技术到商业的思维转化。
- **清晰的业务洞察**：不止于数据呈现，更聚焦于可落地的商业结论。
- **可复现的代码**：提供清晰、注释完整的SQL代码，便于他人理解与验证。
- **专业可视化**：使用Power BI构建交互式看板，直观呈现关键业务指标。

##  核心发现（业务价值总结）

| 分析维度 | 核心结论 | 业务启示 |
| :--- | :--- | :--- |
| **业务模式** | “批发为主，零售为辅”的混合模式。**头部20%批发客户贡献超70%营收**。 | 营收依赖度过高，需建立大客户风险管理与深度绑定机制。 |
| **增长驱动** | 销售额年增 **338%**，客户数年增 **208%**。 | 业务处于爆发期，市场与产品得到验证，应加速扩张。 |
| **客户健康度** | 存在从 **批发大客户（单笔£168K）** 到 **高忠诚零售客户（年购14次）** 的完整梯度。 | 客户结构健康，具备实施精细化分层运营（如会员体系）的坚实基础。 |
| **运营质量** | 整体退货率仅 **1.85%**，远低于行业平均水平（通常5-10%）。 | 供应链与品控出色，是重要的竞争优势，可在营销中强化此点。 |
