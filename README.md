# 零售业务数据分析项目

## 项目概述
通过分析2010-2011年零售交易数据，深度挖掘业务模式、增长驱动和优化机会。

## 核心发现
- **业务模式**：批发+零售混合模式，大宗批发客户单笔消费£168,469
- **增长表现**：年销售额增长338%，客户数增长208%
- **客户结构**：健康的客户梯度，从批发大客户到忠诚零售客户
- **运营质量**：1.85%退货率，远低于行业平均水平

SQL · 数据分析 · Power BI · 业务洞察
