# 零售业务数据分析项目

## 项目简介
这是我的第一个数据分析项目，通过对零售交易数据的分析，学习SQL和业务洞察能力。

## 业务背景
- 分析2010-2011年零售交易数据
- 数据规模：5,000+条交易记录
- 目标：理解业务模式、识别增长机会

## 关键发现
1. **业务模式**：批发为主，平均每单299件商品
2. **增长表现**：年增长率338%
3. **核心客户**：TOP客户年消费16.8万元

## 文件说明
- `sql_analysis.sql` - 完整的SQL分析代码
- `analysis_report.md` - 业务分析报告
- `data/sample_data.csv` - 样本数据
