零售业务数据分析项目
1.项目概述：通过对某零售企业2010-2011年交易数据的分析，识别业务模式、增长趋势和优化机会。
2.业务背景：（1）数据时间范围：2010年1月-2011年12月（2）数据规模：5,000+条交易记录（3）分析目标：理解业务模式、识别增长驱动、发现优化机会
3.分析结论：（1）.业务模式：批发为主，平均每单298.8件商品（2）.增长表现：2011年销售额较2010年增长338%（3）.核心客户：TOP客户年消费16.8万元（4）.商品策略：纸艺产品为主要收入来源
4.文件说明：（1）‘sql/data_analysis.sql’：主要分析代码（2）‘analysis/insights_report.md’：详细分析报告（3）‘data/raw_data.csv’：原始数据
5.技术栈：SQL,Power BI可视化, 业务洞察
